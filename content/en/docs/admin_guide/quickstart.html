Quickstart guide

This topic provides collects basic Katzenpost commands and other information in a single convenient place. All system commands require superuser privileges.

The commands listed here do not apply to the Katzenpost Docker image, which has its own controls. For more information, see Using the Katzenpost Docker test network.

Table 1. 

Task

Resource

Get the Katzenpost server code.

git clone git@github.com:katzenpost/katzenpost.git

Get the Katzenpost chat client code.

git clone git@github.com:katzenpost/katzen.git

View the Katzenpost documentation.

https://katzenpost.network/

Start the mixnet.

systemctl start pq-katzenpost-mixserver

Stop the mixnet.

systemctl stop pq-katzenpost-mixserver

Restart the mixnet.

systemctl restart pq-katzenpost-mixserver

Start mixnet on boot.

systemctl enable pq-katzenpost-mixserver

Disable mixnet start on boot.

systemctl disable pq-katzenpost-mixserver

Control a specific mix server node

Run pq-katzenpost-mixserver -h for options.

Control a specific directory authority

Run pq-katzenpost-authority -h for options.

Monitor the mixnet logs in real time.

journalctl -u pq-katzenpost-mixserver -f -n 2000
